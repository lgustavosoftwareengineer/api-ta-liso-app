⚙️ Como tudo se conecta
1
Usuário acessa o app → Route 53 → CloudFront entrega o Vue.js do S3
2
Vue.js chama a API → Route 53 aponta api.taliso.com.br para o Elastic IP → Nginx recebe na :443 e faz proxy para FastAPI na :8000
3
FastAPI processa → lê/escreve no RDS PostgreSQL (subnet privada) → usa Redis local para tokens de login
4
Login sem senha → token de 6 dígitos salvo no Redis (TTL 10min) → enviado por e-mail via SES (gratuito)
5
Deploy automático → push na main → GitHub Actions SSH no EC2 → git pull + pip install + alembic upgrade head + systemctl restart
